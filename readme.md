# Laravel-test

Companies and Employees

### Installation ###
```sh
* `git clone https://github.com/Maria963/laravel-test.git`
* `cd laravel-test`
* `composer install`
* Create a database laravel-test and inform *.env* 
* `php artisan migrate`
*`php artisan db:seed --class="UsersTableSeeder"`
* `npm install`
* `npm run dev`
* `php artisan serve` to start the app on 
```
